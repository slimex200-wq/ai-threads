# AI 카드뉴스 v3 설계

## 개요

카드뉴스에 기사 썸네일 이미지 추가 + GitHub Pages 다운로드 갤러리 페이지.

## 1. 카드 이미지 추가

### 이미지 소스
1. RSS 기사 URL에서 `og:image` 메타태그 추출 (우선)
2. 실패 시 Unsplash API로 기사 키워드 기반 검색 (폴백)

### 모듈: `image_fetcher.py`
- `fetch_thumbnail(article)` — 단일 기사의 썸네일 가져오기
  - 기사 URL에 HTTP 요청 → `<meta property="og:image">` 파싱
  - 실패 시 Unsplash API 검색 (기사 제목 키워드)
  - 이미지를 120x120으로 리사이즈, 라운드 코너 처리
  - base64 인코딩하여 반환
- `fetch_all_thumbnails(articles)` — 전체 기사 순회
  - 각 기사에 `thumbnail_b64` 필드 추가
  - 이미지 없으면 None (렌더링 시 스킵)

### 의존성 추가
- `requests` — HTTP 요청 (og:image 추출)
- `beautifulsoup4` — HTML 파싱

### Unsplash API
- 무료 티어: 50 requests/hour
- API 키: `UNSPLASH_ACCESS_KEY` 환경변수 / GitHub Secrets
- 엔드포인트: `GET /search/photos?query={keyword}&per_page=1`

## 2. 카드 레이아웃 변경

### 뉴스 카드 수정
- 기존 레이아웃 유지
- 우측 상단 헤더 영역에 120x120 썸네일 추가
- 라운드 코너 (border-radius: 12px)
- 썸네일 없으면 기존 레이아웃 그대로

### CSS 변경
```css
.thumbnail {
    width: 120px; height: 120px;
    border-radius: 12px;
    object-fit: cover;
    flex-shrink: 0;
}
```

### 헤더 영역 변경
- 기존: `번호 | 출처` (양쪽 정렬)
- 변경: `번호 + 출처 | 썸네일` (왼쪽 텍스트 + 오른쪽 이미지)

## 3. GitHub Pages 다운로드 갤러리

### 모듈: `page_generator.py`
- `generate_gallery(output_base_dir)` — docs/index.html 생성
  - output/ 폴더 스캔하여 날짜별 폴더 목록
  - 최신순 정렬
  - 각 날짜별 카드 이미지 그리드 표시
  - 클릭 시 원본 PNG 다운로드

### 디자인
- Vercel 다크 스타일 (카드뉴스와 통일)
- 배경: #0a0a0a, 텍스트: #fff
- 카드 그리드: 3열 (데스크탑), 2열 (모바일)
- 각 카드에 날짜 라벨

### GitHub Pages 설정
- Source: `docs/` 폴더
- 이미지 경로: `../output/YYYY-MM-DD/card-XX.png` (상대 경로)

### 워크플로우 확장
기존 daily-cardnews.yml에 단계 추가:
1. (기존) 카드뉴스 생성
2. (NEW) gallery HTML 생성: `python -c "from page_generator import generate_gallery; generate_gallery('output')"`
3. (기존 수정) `git add output/ docs/` → 커밋 + 푸시

## 4. 파이프라인 변경

```
RSS 수집 (50개)
  → 키워드 필터 (10개)
  → 이미지 수집 (10개에 썸네일 추가)    ← NEW
  → Claude 선별+문구 (4개)
  → 카드 렌더링 (썸네일 포함)            ← MODIFIED
  → 갤러리 HTML 생성                    ← NEW
  → 저장/커밋
```

## 6. 에러 처리

- og:image 추출 실패 → Unsplash 폴백
- Unsplash API 실패 → 썸네일 None (이미지 없이 렌더링)
- UNSPLASH_ACCESS_KEY 없음 → Unsplash 스킵, og:image만 시도
- 갤러리 생성 실패 → 카드뉴스 자체는 정상 저장
